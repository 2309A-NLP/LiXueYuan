五、技术栈一览

层级技术说明
前端React 18 + ViteSPA，SSE 流式渲染，JWT 鉴权
后端框架FastAPI + Uvicorn异步 ASGI，自动 OpenAPI 文档
认证JWT (HS256) + bcrypt无状态 Token 鉴权
关系库MySQL 8.0 + SQLAlchemy用户/角色/会话/消息持久化
缓存Redis 7短期对话记忆，角色槽位管理
向量库Milvus 2.x向量+BM25 双索引，COSINE 度量
LLMDeepSeek-V3对话生成、Query 改写、摘要
Embeddingbge-large-zh-v1.51024维中文向量
Rerankbge-reranker-v2-m3精排重排序
知识图谱Neo4j实体关系三元组
容器化Docker Compose一键部署全部服务
反向代理Nginx静态资源 + API 代理
