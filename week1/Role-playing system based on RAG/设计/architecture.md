## 二、聊天请求处理流程

```
用户发送问题
     │
     ▼
┌──────────────────┐
│ 1. JWT 鉴权      │ ← Authorization: Bearer <token>
└────────┬─────────┘
         ▼
┌──────────────────┐
│ 2. 屏蔽词检测    │ ← blocked_words.py
└────────┬─────────┘
         ▼
┌──────────────────┐
│ 3. 加载短期记忆  │ ← Redis: 最近 20 轮对话
└────────┬─────────┘
         ▼
┌──────────────────┐
│ 4. Query Rewrite │ ← LLM 将多轮对话 + 当前问题改写为独立检索 Query
└────────┬─────────┘
         ▼
┌──────────────────────────────────────────────────┐
│ 5. RAG 混合检索                                   │
│                                                    │
│   ┌─────────────┐    ┌──────────────┐             │
│   │ BM25 关键词 │    │ ANN 向量检索  │  ← 并行执行 │
│   │   (权重0.4)  │    │   (权重0.6)   │             │
│   └──────┬──────┘    └──────┬───────┘             │
│          └────────┬─────────┘                      │
│                   ▼                                │
│          ┌────────────────┐                        │
│          │  融合 + 去重    │                        │
│          └───────┬────────┘                        │
│                  ▼                                 │
│          ┌────────────────┐                        │
│          │ Rerank 精排     │ ← BAAI/bge-reranker   │
│          │ (SiliconFlow)  │   -v2-m3               │
│          └───────┬────────┘                        │
│                  ▼                                 │
│          Top-K 结果 (默认5条)                       │
└──────────────────┬───────────────────────────────────┘
                   ▼
┌──────────────────────────────────────────────────┐
│ 6. 知识图谱增强 (可选)                            │
│    GraphService → Neo4j 子图查询                  │
│    LightRAGService → 图+向量混合                  │
└──────────────────┬───────────────────────────────┘
                   ▼
┌──────────────────┐
│ 7. 构建 Prompt   │ ← 系统人设 + 上下文 + 记忆 + 环境信息(天气/时间)
└────────┬─────────┘
         ▼
┌──────────────────┐
│ 8. LLM 流式生成  │ ← DeepSeek-V3 via SiliconFlow SSE
└────────┬─────────┘
         ▼ (逐 token 推送)
┌──────────────────┐
│ 9. SSE 推送前端  │ ← data: {"delta":"..."}\n\n
└────────┬─────────┘
         ▼
┌──────────────────┐
│10. 持久化消息    │ ← MySQL: messages 表 (问+答)
│   更新记忆       │ ← Redis: 滑动窗口更新
└──────────────────┘
```

## 四、技术栈一览

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | React 18 + Vite | SPA，SSE 流式渲染 |
| 后端框架 | FastAPI + Uvicorn | 异步 ASGI，自动 OpenAPI 文档 |
| 认证 | JWT (HS256) + bcrypt | 无状态 Token 鉴权 |
| 关系库 | MySQL 8.0 + SQLAlchemy | 用户/角色/会话/消息 |
| 缓存 | Redis 7 | 短期对话记忆，角色槽位管理 |
| 向量库 | Milvus 2.x | 向量+BM25 双索引，COSINE 度量 |
| LLM | DeepSeek-V3 (SiliconFlow) | 对话生成、Query 改写、摘要 |
| Embedding | BAAI/bge-large-zh-v1.5 | 1024维中文向量 |
| Rerank | BAAI/bge-reranker-v2-m3 | 精排重排序 |
| 知识图谱 | Neo4j (可选) | 实体关系三元组 |
| 容器化 | Docker + docker-compose | 一键部署全部服务 |
| 反向代理 | Nginx | 静态资源 + API 代理 |
